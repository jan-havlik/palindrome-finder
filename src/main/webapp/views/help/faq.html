What advantages does this software have?

  • We support analysis of long sequences (there is no actual limit, but longer sequences take long time - be patient).
  • You can search for inverted repeats using 4 parameters (length of repeat, spacer length, amount of mismatches and cicularity).
  • We allow you to store results of your analysis for comparison with others. This feature requires user account.
  • We support export of found sequences and summary of stored results in CSV format.
  • Direct NCBI import using sequence ID.

What limitations does this software have?

  • Only 5000 inverted repeats can be displayed at once. It is due to the high computational complexity of filtration and display logic. If you encounter this limitation, we advise you to change parameters of inverted repeat search.
  • There are currently no other limitations, use this software wisely to preserve this policy for other users. Thank you.

Is there any development going on?

We are currently working on new features and bug fixes (2016).
