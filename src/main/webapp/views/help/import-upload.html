Usage of upload

We support currently only 2 types of data inputs - plain text files with A, C, G and T characters encoding the sequence and Fasta format.
When uploading a file, select a name for your sequence. Then select data type (plain or fasta). Choose a file from your hard drive and press Upload button. After a while the file should be available in overview of uploaded/imported sequences - My sequences.

Usage of NCBI

Simply type a NCBI ID (e.g. NC_001770) and press import button. After a while the file should be available in overview of uploaded/imported sequences - My sequences.

Only registered users can upload/import sequences.
